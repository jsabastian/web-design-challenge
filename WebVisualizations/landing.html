Latitude

A Weather Analysis Project

Project Summmary

The data presented here aims to analyze patterns is weather as location nears the equator. The cities that comprise the dataset was randomly assembled via OpenWeatherMap API.

Latitudes were compared to a number of parameters: Maximum Temperatures, Cloudiness, Humidity, and Wind Speed. Data presented within this site include analysis, plot images and the source data for the cities analyzed.
